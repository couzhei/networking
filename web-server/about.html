About This Web Server

This is a simple HTTP web server implementation in Rust.

Technical Details:

  • Language: Rust
  • Port: 8080 (to avoid conflicts with existing web servers on port 80)
  • Protocol: HTTP/1.1
  • Concurrency: Single-threaded (processes one request at a time)

How it works:

  1. Creates a TCP listener socket bound to 127.0.0.1:8080
  2. Accepts incoming client connections
  3. Reads and parses HTTP requests
  4. Determines the requested file path
  5. Attempts to read the file from the file system
  6. Generates appropriate HTTP response (200 OK or 404 Not Found)
  7. Sends the response back to the client
